# Wallet Application

Digital wallet for storing and managing verifiable credentials.

## Features

- Store credentials securely
- Create presentations
- Manage DIDs
- Share credentials via QR code
- Backup and restore

## Tech Stack

- React/Vue/Svelte (TBD)
- State Management: Redux/Vuex/Svelte Store
- UI Library: Material-UI/Tailwind CSS
- Storage: IndexedDB/LocalStorage (encrypted)
- QR Code: qrcode-generator

## Getting Started

```bash
npm install
npm run dev
```
