# Verifier Application

Frontend application for credential verifiers.

## Features

- Verify credentials
- Request credential presentations
- View verification results
- Scan QR codes for credentials
- Check revocation status

## Tech Stack

- React/Vue/Svelte (TBD)
- State Management: Redux/Vuex/Svelte Store
- UI Library: Material-UI/Tailwind CSS
- QR Code: qrcode-reader

## Getting Started

```bash
npm install
npm run dev
```
